Stock Price Prediction Using Deep Learning Algorithms Based on Technical Indicators

논문이 사용한 실험 방식 재현표

| 구분                 | 내용 요약                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **연구 대상 종목/데이터셋**  | - 터키 이스탄불 증권거래소 BIST100 상장 종목 중 시가총액 상위 5개<br>- 종목: Koç Holding (KCHOL.IS), Turkish Airlines (THYAO.IS), Ford Otosan (FROTO.IS), Türkiye Petrol Rafinerileri (TUPRS.IS), İş Bankası (ISCTR.IS)<br>- 기간: 2020.02 \~ 2023.12<br>- 일일 종가 데이터 (Yahoo Finance에서 수집)                                                                                                                                                                                                                                                        |
| **원시 피처**          | - 일일 종가<br>- 거래량<br>- 시장 시가총액 기준으로 상위 5개 종목 선택                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **사용 기술지표 및 파라미터** | - CUMLOGRET\_1 (누적 로그 수익률)<br>- EMA\_8 (지수이동평균)<br>- SMA\_8 (단순이동평균)<br>- MACD 계열 (MACD\_8\_21\_9, MACDh\_8\_21\_9, MACDs\_8\_21\_9)<br>- FWMA\_8 (피보나치 가중 이동평균)<br>- Bollinger Bands (BBL\_5\_2.0, BBM\_5\_2.0, BBU\_5\_2.0, BBB\_5\_2.0, BBP\_5\_2.0)<br>- Stochastic RSI (STOCHRSIk\_14\_14\_3\_3, STOCHRSId\_14\_14\_3\_3)<br>- Stochastic Oscillator (STOCHk\_14\_3\_3, STOCHd\_14\_3\_3)<br>- Momentum (MOM\_10)<br>- Ichimoku (ISA\_9, ISB\_26, ITS\_9, IKS\_26, ICS\_26)<br>- KDJ (K\_9\_3, D\_9\_3, J\_9\_3) |
| **전처리**            | - 결측치 제거<br>- 로그 수익률 계산<br>- 정규화 (스케일링)<br>- 상관계수 분석 후 0.8 이상 상관되는 지표만 선택                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **데이터 분할**         | - 학습 : 70%<br>- 테스트 : 30%<br>- 윈도잉 길이는 명시 안됐지만 250일 구간 예측 그림 제시                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **사용 모델 구조**       | ① **Single-Layer LSTM**<br> - 1층 LSTM (64 neurons, relu)<br> - Adam, MSE, 100 epochs, batch size=8<br><br>② **Three-Layer LSTM**<br> - 64-128-128 neurons<br> - Activation: relu, tanh, tanh<br> - 동일 옵티마이저/하이퍼파라미터<br><br>③ **Three-Layer BiLSTM**<br> - 64-128-128 neurons<br> - Activation: relu, tanh, tanh<br><br>④ **CNN-LSTM (Hybrid)**<br> - CNN layer로 피처 추출 후 LSTM 연결 (세부 파라미터 논문 본문에 없음)                                                                                                                   |
| **평가 지표**          | - MAE, MAPE, MSE, MSLE, RMSE, EVS, R² score, CPU time                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **성능 요약 (주요 결과)**  | - 가장 낮은 MAPE와 RMSE는 **Single-Layer LSTM**에서 관찰<br>- 예) FROTO.IS 결과<br> • RMSE = 0.0266<br> • MAPE = 4.82%<br> • R² = 0.9865<br>- 다른 종목에서도 단일 LSTM이 전반적으로 가장 좋은 결과<br>- BiLSTM/CNN-LSTM은 일부 종목에서 더 느리고 RMSE/MAPE 높음                                                                                                                                                                                                                                                                                                    |
| **주요 결론**          | - 기술지표를 통한 노이즈 제거가 예측성능 향상에 기여<br>- 단일 LSTM 구조가 오히려 다층/양방향/CNN-LSTM보다 과적합 없이 더 안정적 예측<br>- 다양한 딥러닝 아키텍처 비교 실험으로 실무적 가이드 제공                                                                                                                                                                                                                                                                                                                                                                                            |
