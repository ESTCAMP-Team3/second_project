- `0630_model3_base.ipynb`:
    ### 📌 Baseline 모델링 결과 (Apple 종가 예측)

    ✅ 입력 데이터

    - Apple 5년치 일별 OHLCV

    - 윈도우 크기: 20일

    - 타겟: 다음날 종가 (스케일링 없이 원 단위 그대로)

    ✅ 모델 구조

    - LSTM(48 units) + Dense(1)

    - X만 StandardScaler 적용

    - Loss: MAE

    ✅ 성능 지표 (Test set 2022년)

    - MAE ≈ 3.2달러
    
    - RMSE ≈ 4.0달러

    - R² ≈ 0.90

    ✅ 인사이트

    - OHLCV만으로도 단일스텝 가격을 2~3% 오차 수준으로 예측

    - 타겟 스케일링 없이 가격 단위로 바로 출력 → 서비스 적용 용이
    
    - 프로젝트 내 **baseline**으로 채택하고, 이후 기술지표 추가, horizon 확장, 모델 구조 개선 등 개선 실험 계획
